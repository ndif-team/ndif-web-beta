# ndif-website

The website for the [National Deep Inference Fabric](https://ndif.us/) ([ndif.us](https://ndif.us/)).

## Installation
- Install `bun` (https://bun.com/)
- run `bun install`

## Running NDIF.us
- `cd packages/ndif.us`
- `bun run dev`

## How to push to prod
Pushing to main automatically pushes the entire website and builds it and runs it in Github pages! Voila.
