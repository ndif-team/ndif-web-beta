# ndif-website

The website for the [National Deep Inference Fabric](https://ndif.us/) ([ndif.us](https://ndif.us/)).

## Installation
- Install `bun` (https://bun.com/)
- run `bun install`

## Running NDIF.us
- `cd packages/ndif.us`
- `bun run dev`
- go to <http://localhost:3000/ndif-web-beta>. Note: When this beta site goes live we will change it from a /ndif-web-beta url, to a base url like ndif.us.

## How to push to prod
Pushing to main automatically pushes the entire website and builds it and runs it in Github pages! Voila.
